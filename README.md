## 📂 Projects

### 🚗 Vehicle Inspection AI Reporter *(In Progress)*
> Upload a vehicle image → YOLOv8 detects damage and classifies severity → Gemini generates a structured professional inspection report → Streamlit dashboard with downloadable report

**Stack:** `YOLOv8` `Gemini API` `LangChain` `Streamlit` `OpenCV` `Python`

---

### 📄 [PDF RAG Assistant](https://github.com/NavadeepReddy21/PDF-RAG-Assistant)
> Upload any PDF and ask questions in natural language. LangChain splits document into chunks, ChromaDB stores embeddings, Gemini answers with source-page citations and conversation memory.

**Stack:** `LangChain` `ChromaDB` `Gemini API` `HuggingFace Embeddings` `Streamlit`

---

### 👥 Gender Classification & Passenger Counting System
> Real-time edge AI system deployed on Raspberry Pi for RTC buses. Detects and counts male/female passengers using virtual entry/exit line logic from live camera streams.

**My contribution:** Data collection, image annotation, entry/exit line logic code, pipeline testing and validation on edge hardware

**Stack:** `OpenCV` `ONNX Runtime` `Raspberry Pi` `Python`

---

### 🧠 Explainable AI Brain Tumor Diagnostic System
> CNN model using VGG16 transfer learning for multi-class MRI brain tumor classification. Grad-CAM integrated to highlight regions influencing predictions — improving interpretability for medical AI.

**Stack:** `TensorFlow` `Keras` `VGG16` `Grad-CAM` `OpenCV`

---

### 🚘 Auto Vehicle Inspection System (AVIS)
> Custom YOLOv8 model trained on annotated vehicle dataset for real-time component and defect detection. Handled complete ML pipeline — data collection, annotation via Roboflow, model training, evaluation, and OpenCV integration.

**Stack:** `YOLOv8` `OpenCV` `Roboflow` `Python`

---

### 🐾 Automated Pet Care System
> IoT system for automated pet feeding and water dispensing. ESP8266 controls servo motors and pumps based on IR and water-level sensor inputs with real-time monitoring via Firebase and mobile app.

**Stack:** `ESP8266` `Firebase` `Arduino` `Python`
